### Task 2: Make Workspace Unit Tests Deterministic

**Files:**
- Modify: `package.json`
- Test: root `pnpm test`

- [ ] **Step 1: Preserve the observed red evidence**

The pre-change Docker root suite has already reproduced API and UI failures
under concurrent package execution, while the same UI files pass when run
alone. Treat that output as the regression evidence for resource contention.

- [ ] **Step 2: Serialize package-level test scripts**

Change the root script from:

```json
"test": "node scripts/run-pnpm-test.mjs -r --if-present test"
```

to:

```json
"test": "node scripts/run-pnpm-test.mjs -r --workspace-concurrency=1 --if-present test"
```

This keeps each package's Vitest behavior intact and only prevents API, web,
worker, and shared-package suites from competing at the same time.

failures.
